## Mindcraft
A collaborative study platform designed to help students learn more effectively, whether studying solo or with friends.
About
Mindcraft is a web-based study environment that combines productivity tools with social features to create an engaging learning experience for high school and university students.
Features

Study Sessions - Timed focus sessions with break reminders
Game Elements - Gamified learning to keep students motivated
Statistics & Progress Tracking - Monitor study time and achievements
Note Taking - Built-in note editor for quick capture
Real-time Chat - Communicate with study partners
Friend System - Connect with classmates and study buddies
Responsive Design - Works seamlessly on desktop and mobile
